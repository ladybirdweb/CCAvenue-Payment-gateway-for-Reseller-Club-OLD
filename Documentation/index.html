1. Description

This extends Reseller Club to accepts money/payments through CCAvenue Payment gateway on your supersite. South Asia's biggest Payment Gateway and Merchant Accounts service provider.

Plugin is a set of files, with step by step documentation.

Open the documentation, make slight changes, enter API/key values from CCAvenue & Reseller Club account to the files as shown in documentation. Start accepting payment on your store front within 5 to 10 minutes.

Step by step video guide to be uploaded soon.

2. Reseller Club Configuration

Here are the steps/changes that you need to do, to add CCAvenue Payment gateway to Reseller Club

 

  1. Login to Reseller Club
  2. Go to Settings >> Finance & Billing >> Payment Gateway >> List/Add
  3. Click Add a Gateway
  4. Click Add any other Payment Gateway
  5. Give Name to the gateway
    Enter URL of the domain/server where you are hosting this project e.g http://www.yourdomain.com/directory/checkout.php
  6. Go to Settings >> Finance & Billing >> Payment Gateway >> List/Add
    Click Manage on the gateway name that you just created
    Click Generate Key & new key will get generated, save this key, we will use it, in next step

3. CCAvenue Configuration

Here are the steps/changes that you need to do, on CCAvenue website to integrate it with Reseller Club

  1. Login to CCAvenue
  2. Go to Setting >> API Keys, We will use these keys in the next step, in the screenshot API keys are hidden for security reason

4. GUI Web Installer

  • Open http://www.yourdomain.com/directory/install Click Create a Configuration file
  • Click Let's go!
  • Fill in the details & click submit button, directory gets filled automatically

    Thats it, this will create config files and you are ready to go

5. Manual Install

  • Rename config-sample.php to config.php
  • open config.php add the keys from CCAvenue & Reseller Club, which were obtained in earlier steps. Edit the links as shown in screenshot
  • Delete the install folder & all the files inside it
